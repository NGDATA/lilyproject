Updates the properties of the rowlog.
- Respect order : if the order of the subscriptions should be respected or not
- Notify enabled : if the rowlog processor should be notified when messages are put on the rowlog shard 
- Notify delay : the minimal time (in ms) between sending notifications to the rowlog processor
- Minimal process delay : the minimal age (in ms) a message should have before it is eligible to be processed
- Wakeup delay : the time (in ms) the rowlog processor waits to check if there are messages on the rowlog shard if it didn't receive any notifications in the meanwhile
