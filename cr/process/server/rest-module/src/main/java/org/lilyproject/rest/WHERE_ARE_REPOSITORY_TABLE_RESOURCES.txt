Additional JAX-RS resource classes are generated by an annotation processor
reacting on these annotations:

@GenerateTableResource -> creates /table/{tableName}/...
@GenerateRepositoryResource -> creates /{repositoryName}/...
@GenerateRepositoryAndTableResource -> creates /{repositoryName}/table/{tableName}/...

This annotation processor is specified in the configuration of the maven-compiler-plugin,
see the pom.xml.
